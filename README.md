# Heavymetal.dk

> The Danish [Heavymetal.dk](http://heavymetal.dk) website.

## Docker instructions

``` bash
# Run webserver and database on localhost:80
docker-compose up

# Stop webserver and database
docker-compose stop

# Run webserver and database and force rebuild
docker-compose up -d --build
```
